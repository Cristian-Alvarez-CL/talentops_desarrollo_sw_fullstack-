# Instrucciones de Uso

## Instalar dependencias

```bash
npm install commander
```

## Ejecutar demostración completa:

```bash
npm run demo
```

## Usar la interfaz CLI:

```bash
# Inicializar sistema
npm run cli -- inicializar

# Procesar archivos
npm run cli -- procesar ./mis-archivos

# Buscar contenido
npm run cli -- buscar "Node.js"

# Ver estadísticas
npm run cli -- estadisticas

# Gestionar backups
npm run cli -- backup --listar
```

## Ejecutar tests:

```bash
npm test
```
